# PM/PL 바이브 코딩 교육자료

PM(Product Manager)과 PL(Project Leader)을 위한 AI 바이브 코딩 교육자료입니다.

## 교육 모듈

| 모듈 | 주제 | 내용 |
|------|------|------|
| 1 | AI 기초 개념 | LLM, 프롬프트 엔지니어링, RAG, AI 에이전트, 용어 사전 |
| 2 | AI 활용 문서 작성 | PRD, 기획서, 요구사항 정의서, 회의록, PL 프로젝트 관리 문서 |
| 3 | AI 도식화 활용 | 플로우차트, 아키텍처 다이어그램, ER 다이어그램, 사용자 여정 맵 |
| 4 | AI 자료 조사 활용 | 시장 조사, 경쟁사 분석, 페르소나 생성, 데이터 수집 |
| 5 | 바이브 코딩 실습 | Claude Code, 프롬프트로 웹페이지 만들기, 실전 프로젝트 |
| 6 | PM vs PL 역할 이해 | 역할 비교, PL 핵심 역량, WBS, 리스크 관리, 팀 리딩 |

## 기술 스택

- HTML5 / CSS3 / JavaScript
- Mermaid.js (다이어그램)
- Prism.js (코드 하이라이팅)

## 실행 방법

```bash
# 로컬에서 열기
open index.html

# 또는 Live Server 사용
npx live-server
```

## 주요 기능

- 사이드바 목차 네비게이션
- 다크모드 / 라이트모드 전환
- 반응형 디자인 (모바일 대응)
- Mermaid 다이어그램 렌더링
- 코드 블록 Syntax Highlighting
